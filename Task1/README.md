# Task 1: Exploring and Visualizing a Simple Dataset

## Objective
Learn how to load, inspect, and visualize a dataset to understand data trends and distributions.

## Dataset
- Iris Dataset (CSV format, can be loaded via seaborn or downloaded)

## Instructions
- Load the dataset using pandas
- Inspect dataset using `.head()`, `.info()`, and `.describe()`
- Visualize data:
  - Scatter plots for feature relationships
  - Histograms for value distributions
  - Box plots to identify outliers
- Use matplotlib and seaborn for plotting

## Key Observations
- Petal length and width are strongly correlated
- Sepal length and width show moderate correlation
- Boxplots reveal a few outliers in sepal width
